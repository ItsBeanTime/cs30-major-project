# Major Project Proposal

# Description
For my major project I would like to recreate part of the game Undertale. Undertale has a lot of unique concepts such as playing the game as a pacifist or play as a killer. The games combat system is also very interesting, it involves the player to pick an option from, attack, item, act or spare, and after that choice you have to dodge a barage of attacks until you can pick again. This game has a scrolling camera that follows the player along with houses/stores that you can enter. The map is on a pixel grid so it wont be too hard to make.
